Task 2 Upgrade - Room & Premium UI

New Features:
- Room Database: Notes now persist permanently using Room, DAO, and Repository pattern.
- Premium UI: Redesigned with a dark charcoal theme (#0F0F17) and soft pastel cards.
- Staggered Grid: Notes are displayed in a 2-column staggered grid for a modern look.
- ID & Timestamp: Each note card now shows its database ID (#1, #2...) and the exact time it was created.
- Edit/Update: Tapping a card opens a Modal Bottom Sheet to edit existing notes.
- Animations: Smooth entrance and content size animations.

Architecture:
- Follows clean MVVM architecture.
- ViewModel uses StateFlow to reactively update the UI from the Room Flow.
- All database operations are asynchronous using Coroutines.
- Repository acts as a single source of truth.
